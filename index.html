The database for HuggingFace users willing to improve interpretability.

Built to store text, paragraphs, embedded content, entities, tags and relations in a single database, NucliaDB is the database for data-scientists and developers looking to improve the interpretability of their models.

Why a database for NLP applications?

NucliaDB is a high performance database that enables you to save time by exploring text datasets with visual search and text retrieval, and lets you improve the interpretability of your models. Specifically focused on NLP applications, NucliaDB is a distributed production-ready database that scales.

Install NucliaDB locally very easily:

pip install nucliadb
nucliadb --maindb ./main --blob ./blob --node ./node --log=INFO docker run -P -e LOG=INFO -it nuclia/nucliadb:latest
